Tutorial Guide

  • *
    Move the Start Node or the Goal Node, by pressing the mouse button on it and then dragging it to wherever you want.
  • *
    Drag over the Nodes while keeping the mouse button pressed to draw Walls.
  • *
    Select a Pathing or Maze algorithm from the option lists and click on Visualize button to see the algorithm in action.
  • *
    After a Pathing algorithm has finished running, move the Start Node or the Goal Node to instantly see different paths.
